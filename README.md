# Quiz App Colaborativo com Firebase 📱

Este é um aplicativo de quiz moderno para Android, desenvolvido como trabalho prático para a disciplina de Programação para Dispositivos Móveis da Universidade Federal de Uberlândia (UFU). O app utiliza Firebase para autenticação e sincronização de dados em nuvem, garantindo uma experiência fluida tanto online quanto offline.

## 👥 Equipe e Papéis

*   **André:** Desenvolvimento de Backend e Integração com Firebase (Auth, Firestore).
*   **Gustavo:** Planejamento, Desenvolvimento Geral e Correção de Erros (Sincronização de dados e Lógica de Quiz).
*   **Marcelo:** Quality Assurance (QA), Documentação, Verificação de Requisitos e Ajustes Finais.

## 🚀 Funcionalidades Principais

1.  **Login e Autenticação:** Acesso individual via Firebase Authentication (E-mail/Senha).
2.  **Sincronização Offline:** As questões são baixadas do Cloud Firestore e armazenadas localmente via Room Database para uso sem internet.
3.  **Quiz Dinâmico:** Interface interativa para responder perguntas com feedback imediato.
4.  **Histórico e Estatísticas:** Acompanhamento do desempenho individual, com pontuação, percentual de acertos e tempo.
5.  **Ranking e Dashboard:** Visualização dos melhores colocados e do progresso geral do usuário.

## 🛠️ Tecnologias Utilizadas

*   **Linguagem:** Kotlin
*   **Interface:** Jetpack Compose (Material Design 3)
*   **Banco de Dados Local:** Room Database
*   **Backend:** Firebase (Authentication, Cloud Firestore)
*   **Arquitetura:** MVVM (Model-View-ViewModel) com Clean Architecture.
